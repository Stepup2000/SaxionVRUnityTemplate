# SaxionVRUnityTemplate
This repository contains a Unity-based VR starter template, developed as a foundation for creating VR applications and games within Saxion University of Applied Sciences.
The project provides a ready-to-use base environment, allowing developers to start building immediately without going through the full initial setup process.

Features:

- Preconfigured Unity 6 project
- VR interaction system (Movement, far cast / ray + poke interaction etc)
- Hand tracking with pose detection
- Smooth switching between controllers and hand input
- Improved interaction hitboxes for more accurate input
- UI interaction supporting multiple input types
- Ready-to-use scene template (VR rig, player, audio, scripts)
- Structured and organized project layout
- Basic interaction audio feedback
- Preconfigured build settings
- Debugging and logging utilities
- General performance and usability improvements

Purpose:

This template is intended as a starting point for VR development.
It is primarily designed for educational use, but can also serve as a base for prototypes or small-scale projects.

Usage:

You are free to use and modify this project as needed.
If you plan to build on this template:

Fork the repository before making changes
- Treat this repository as a base/template, not as your main project
- Keep your own implementation separate
